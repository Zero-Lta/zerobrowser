# Assets

Esta pasta deve conter os ícones do aplicativo.

## Ícones Necessários

- `icon.png` - Ícone principal (512x512px recomendado)
- `icon.ico` - Ícone para Windows (formato .ico)

## Nota

Por enquanto, o projeto funcionará mesmo sem os ícones, mas para uma experiência completa, adicione os ícones nesta pasta.

Você pode usar ferramentas online como:
- https://www.favicon-generator.org/
- https://icoconvert.com/

Para converter imagens PNG para ICO.
