Privacidade

A tua privacidade
não é negociável.

Somos um browser desktop, não uma operação de recolha de dados. Eis o que isso significa na prática.

TL;DR

  • Zero telemetria. Não enviamos nenhum dado para servidores nossos. Nunca.
  • Zero contas. Não há registo, login ou sincronização na cloud.
  • Dados locais. Histórico, favoritos, passwords — tudo fica guardado na tua máquina.
  • Código aberto. Podes verificar tudo no GitHub.

Que dados recolhemos?

Nenhuns. O Zero Browser não tem servidor de análise, não liga a APIs de telemetria, não envia relatórios de crash automaticamente.

Onde ficam os dados?

Exclusivamente no teu computador, na pasta de dados do utilizador do sistema:

  • Windows: %APPDATA%\ZeroBrowser\

Isto inclui histórico, favoritos, passwords, notas, to-dos, permissões de sites, downloads e cache.

Bloqueio de trackers

Por defeito, bloqueamos pedidos de rede para mais de 70 domínios de tracking conhecidos (Google Analytics, Facebook Pixel, DoubleClick, etc.). O contador na barra mostra quantos foram bloqueados na página atual.

Auto-update

O browser verifica se há novas versões em api.github.com (GitHub Releases). Este é o único pedido automático que o próprio Zero Browser faz para o exterior. Podes desativá-lo nas definições.

Extensões

Quando instalas uma extensão, ela corre com as suas próprias permissões. Nós não controlamos o que extensões de terceiros fazem — revê-as cuidadosamente.

Sites que visitas

Os sites que abres podem tentar recolher dados sobre ti (cookies, fingerprinting, etc.). O nosso bloqueio de trackers reduz isto, mas não elimina. Usa Janela Privada para sessões verdadeiramente isoladas.

Passwords

São guardadas localmente no ficheiro passwords.json. Recomendamos o uso adicional de um password manager dedicado para credenciais sensíveis.

Atualizações a esta política

Alterações serão publicadas nesta página e no repositório GitHub. A versão em vigor é sempre a mais recente.
